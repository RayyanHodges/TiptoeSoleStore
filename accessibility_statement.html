Accessibility Statement for TipToe Sole Store

This is an accessibility statement from TipToe Sole Store.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. TipToe Sole Store is fully conformant with WCAG 2.1 level AA. Fully conformant means that the content fully conforms to the accessibility standard without any exceptions.

TipToe Sole Store assessed the accessibility of TipToe Sole Store by the following approaches:

  • Self-evaluation
  • Using services such as Browserling (https://www.browserling.com/) to test individual browser cross compatibility
  • Apps like Colour Contrast Analyzer (CCA) to check colour contrast with text and background page colours
